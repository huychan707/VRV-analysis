# VRV-analysis
Welcome to VRV-analysis, a specialized tool for analyzing the operational data of VRV air conditioning systems.
Developed by Trương Nguyễn Quang Huy, this website utilizes data extracted from Marutto to provide a deep and detailed insight into the performance of VRV systems.
With an intuitive interface and in-depth analytical charts, VRV-analysis helps users easily monitor, evaluate, and optimize the operation of their VRV systems for maximum efficiency.
